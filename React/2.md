## 1、属性的介绍与用法
------------
### 1.1 属性概念
- props = properties 
- 一个事物的性质与关系，与生俱来的，往往无法改变
### 1.2 属性的用法
```jsx
// 直接设置的形式
<HellWorld name="text">
<HellWorld name={"text"}>
<HellWorld name={123}>
<HellWorld name={[1, 2, 3]}>
<HellWorld name={variable}>
```
```jsx
// setProps 方法设置（用的比较少）
setProps({123});
```
```jsx
var props = {
    one: '123',
    two: 321,
    tnree: ['Alan', 'Ben']
}
// 这里的 ... 代表拿到props下的所有对象
<HelloWorld {...props}>
```

```jsx
"use strict";
class List extends React.Component {

        const commentNodes = this.props.name.map(function(dataValue, dataKey) {
            return <li key={dataKey}>{dataValue}</li>
        });
        return (<ul>{commentNodes}</ul>);
    }
}
```

设置**默认**props例子,下面将其设置默认是string
```jsx
var Hello = React.createClass({
    getDefaultProps: function() {
        return {
            name: 'Alan',
        };
    },
    propTypes: {
        name: React.PropTypes.string,
    },
    render: function() {
        return (
            <div>{this.props.name}</div>
        );
    }
});

ReactDOM.render(
    <Hello />,
    document.getElementById('root')
);
-------------------------------------------------
```
